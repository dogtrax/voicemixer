CLMOOC Poem Voice Mixer

The CLMOOC community is an open, collaborative affinity network with global reach that invites and encourages participants to make art, in whatever form that is. In this experimental project, we are using an original poem and a multitude of CLMOOC voices for a remix activity. You are invited to follow and engage with CLMOOC activities through Twitter and elsewhere.

Start by making a mix...

For the Mixer The text below can be saved to a file voices.txt and used by a mixmaster who has all the source audio to run via command line. Try downloading a version.
